# 🔗 二维码链接修复报告

**修复时间**: 2026-03-24 15:42 GMT+8  
**问题**: 二维码包含 localhost 链接，用户扫码无法访问  
**状态**: ✅ 已修复

---

## 🐛 问题描述

用户反馈：二维码包含的注册链接是 `http://localhost:3011/register?ref=xxx`，用户扫码后无法打开。

**原因**: 
- 后端代码使用了默认值 `process.env.BASE_URL || 'http://localhost:3011'`
- 生成二维码时没有使用正确的公网域名

---

## ✅ 修复方案

### 1. 修改后端代码

**文件**: `backend/src/controllers/authController.js`

**修改内容**:
```javascript
async function generateQRCodeBase64(user) {
  // 使用移动端前端地址（Vercel 部署）
  const frontendUrl = process.env.FRONTEND_URL || 'https://distribution-system-psi.vercel.app';
  const registerUrl = `${frontendUrl}/register?ref=${user.referralCode}`;
  
  console.log(`为用户 ${user.username} 生成二维码，注册链接：${registerUrl}`);
  
  const qrCodeDataUrl = await QRCode.toDataURL(registerUrl, {
    width: 300,
    margin: 2
  });
  
  return qrCodeDataUrl;
}
```
