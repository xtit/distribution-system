## 📱 移动端截图指南

### 截图尺寸建议
- **宽度**: 375px (iPhone 宽度)
- **高度**: 667px 或根据内容自适应
- **格式**: PNG
- **质量**: 高质量

### 截图方法

#### 方法 1: 浏览器开发者工具

1. 打开移动端：https://distribution-system-psi.vercel.app
2. 按 `F12` 打开开发者工具
3. 点击设备模拟按钮（或按 `Ctrl+Shift+M`）
4. 选择设备：iPhone 12 Pro 或类似尺寸
5. 访问各个页面并截图

#### 方法 2: 手机实际截图

1. 在手机上打开移动端
2. 访问各个页面
3. 使用手机截图功能
4. 传输到电脑并重命名

### 需要截图的页面

#### 1. 登录页面 (`mobile-login.png`)
- 显示用户名、密码输入框

## 🎨 截图优化建议

### 1. 数据准备
- 确保有测试数据（商品、订单、用户等）
- 数据要有代表性（不同状态）
- 避免敏感信息

### 2. 截图时机
- 页面完全加载后
- 避免 loading 状态
- 避免弹窗干扰

### 3. 图片处理
- 可以适当裁剪空白区域
- 保持清晰度
- 统一尺寸风格
- 压缩文件大小（建议 < 500KB/张）

### 4. 隐私保护
- 打码敏感信息（手机号、邮箱等）
- 使用测试数据
- 避免真实用户信息

## 📤 提交截图

### 方法 1: Git 提交

```bash
# 将截图文件放到 docs/screenshots/ 目录
git add docs/screenshots/*.png
git commit -m "添加系统截图"
git push
```

### 方法 2: GitHub 上传

1. 访问 GitHub 仓库
2. 进入 `docs/screenshots/` 目录
3. 点击 "Upload files"
4. 选择截图文件
5. 提交更改
