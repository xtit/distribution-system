## ✅ 已完成的功能

### 1. 上架/下架功能
- ✅ 每个商品后显示"上架"或"下架"按钮
- ✅ 点击按钮弹出二次确认
- ✅ 确认后切换商品状态
- ✅ 状态实时更新

**按钮逻辑**:
```
商品状态 = 上架 → 显示"下架"按钮（黄色）
商品状态 = 下架 → 显示"上架"按钮（绿色）
```

### 2. 删除二次确认
- ✅ 点击删除弹出详细确认框
- ✅ 显示商品名称
- ✅ 说明删除后移入回收站
- ✅ 可取消操作

**确认框文案**:
```
⚠️ 确定要删除商品"XXX"吗？

删除后将移入回收站，可从回收站恢复或彻底删除。

[取消] [确定]
```

### 3. 回收站功能
- ✅ 点击"回收站"按钮查看已删除商品
- ✅ 显示回收站商品列表
- ✅ 支持恢复单个商品
- ✅ 支持清空回收站（彻底删除）

**回收站操作**:
```
回收站商品:
1. 商品 A (¥99)
2. 商品 B (¥199)

输入操作:
- 输入商品 ID 恢复该商品
- 输入 "clear" 清空回收站
- 留空取消
```

### 4. 删除逻辑
- ✅ 软删除：status = 0（下架）
- ✅ 不是真正从数据库删除
- ✅ 可从回收站恢复
- ✅ 恢复后 status = 1（上架）

  );
  if (!confirmed) return;

## 📝 操作流程

### 上架/下架流程
```
1. 进入"商品管理"页面
2. 找到目标商品
3. 点击"上架"或"下架"按钮
4. 确认操作
5. 状态立即更新
```

### 删除商品流程
```
1. 进入"商品管理"页面
2. 点击商品的"删除"按钮
3. 阅读确认框信息
4. 点击"确定"
5. 商品移入回收站
```

### 恢复商品流程
```
1. 点击"回收站"按钮
2. 查看回收站商品列表
3. 输入要恢复的商品 ID
4. 确认恢复
5. 商品重新上架
```
