## 🚀 操作流程示例

### 订单发货流程
1. 进入"订单管理"标签页
2. 点击订单的"详情"按钮
3. 查看订单详细信息
4. 点击"发货"按钮
5. 填写收货人信息（姓名、手机号、地址）
6. 确认发货，订单状态变为"已发货"

### 佣金发放流程
1. 进入"佣金管理"标签页
2. 查看佣金统计卡片
3. 找到状态为"已确认"的佣金
4. 点击"发放"按钮
5. 确认发放，佣金状态变为"已发放"

### 商品编辑流程
1. 进入"商品管理"标签页
2. 点击商品的"编辑"按钮
3. 弹窗自动填充商品信息
4. 修改需要更新的字段
5. 点击"保存"
6. 商品列表自动刷新
