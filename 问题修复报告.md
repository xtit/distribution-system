# 分销系统 - 问题修复报告

**修复时间**: 2026-03-19 19:00 (GMT+8)

## 🔧 已修复的问题

### 1. 后端 API 错误

#### 问题 1: Commission 模型未导入
**错误**: `ReferenceError: Commission is not defined`
**位置**: `backend/src/controllers/authController.js`
**修复**: 添加 Commission 模型导入
```javascript
const { User, Referral, Commission, Product, Order } = require('../models');
```

#### 问题 2: Sequelize 字段映射错误
**错误**: `SQLITE_ERROR: no such column: order.createdAt`
**位置**: `authController.js` - getCommissions 函数
**修复**: 使用正确的数据库字段名
```javascript
attributes: ['id', 'orderNo', 'status', 'actualAmount', 'created_at']
order: [['created_at', 'DESC']]
```

#### 问题 3: 二维码生成路径错误
**问题**: 二维码文件生成在错误的目录
**修复**: 使用 `process.cwd()` 获取正确路径
```javascript
const qrCodeDir = path.join(process.cwd(), 'qrcodes');
```

#### 问题 4: 注册链接 URL 错误
**问题**: 移动端注册链接使用了错误的端口
**修复**: 更新为正确的移动端 URL
```javascript
const baseUrl = process.env.BASE_URL || 'http://localhost:3011';
const registerUrl = `${baseUrl}/register?ref=${user.referralCode}`;
```

### 2. 前端错误

#### 问题 1: copyText 导出错误
**错误**: `The requested module does not provide an export named 'copyText'`
**位置**: `mobile-frontend/src/views/Profile.vue`, `QRCode.vue`
**修复**: 使用原生 Clipboard API
```javascript
// 修复前
import { copyText } from 'vant'
copyText(user.value.referralCode)

// 修复后
navigator.clipboard.writeText(user.value.referralCode)
```
