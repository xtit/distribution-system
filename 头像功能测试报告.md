## 🐛 可能的问题

### 问题 1: 头像不显示

**检查**:
1. 后端服务是否运行
2. Vite 代理配置是否正确
3. 图片 URL 是否正确
4. 浏览器控制台是否有错误

**解决**:
```bash
# 重启后端
cd backend
node src/index.js

# 重启前端
cd mobile-frontend
npm run dev
```

### 问题 2: 保存后不更新

**检查**:
1. updateProfile 是否返回 user 数据
2. userStore 是否正确更新
3. 页面是否重新渲染

**解决**:
- 确保使用展开运算符：`userStore.user = { ...res.data.user }`
- 刷新页面验证
