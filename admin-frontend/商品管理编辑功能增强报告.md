### 3. 修改主页 (`index.html`)

#### 编辑功能
```javascript
// 旧代码：弹窗显示简单信息
function editProduct(id) {
  showModal('编辑商品', '编辑功能开发中...', () => {});
}

// 新代码：新标签页打开编辑页面
function editProduct(id) {
  const editUrl = `product-edit.html?id=${id}`;
  window.open(editUrl, '_blank');
}
```

### 4. 后端 API 增强

#### 新增接口

**获取商品详情**
- 路由：`GET /api/admin/products/:id`
- 控制器：`adminController.getProductDetail`
- 用途：编辑页面加载商品数据

**图片上传**
- 路由：`POST /api/admin/upload`
- 控制器：`adminController.uploadImage`
- 中间件：multer
- 用途：商品图片上传
- 返回：`{ url: '/uploads/image-xxx.jpg' }`

#### 新增文件

**上传中间件** (`backend/middleware/upload.js`)
- 配置 multer 存储
- 文件类型过滤（仅图片）
- 文件大小限制（5MB）
- 唯一文件名生成

**路由修改** (`backend/src/routes/admin.js`)
- 添加商品详情路由
- 添加图片上传路由
- 引入 upload 中间件

**控制器修改** (`backend/src/controllers/adminController.js`)
- 添加 `getProductDetail` 函数
- 添加 `uploadImage` 函数

## 🚀 使用方式

### 启动后端
```bash
cd /home/admin/openclaw/workspace/distribution-system/backend
npm start
```

### 访问管理后台
1. 打开浏览器访问：`http://localhost:3010/admin-frontend/index.html`
2. 登录后进入商品管理页面
3. 点击 **"➕ 添加商品"** 或 **"✏️ 编辑"** 按钮
4. 在新标签页中编辑商品信息
5. 保存后自动关闭，返回商品列表刷新

## 🔧 技术栈

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **后端**: Node.js + Express
- **文件上传**: Multer
- **数据库**: SQLite (Sequelize ORM)

## 📝 注意事项

1. **图片上传路径**: 图片保存在 `backend/uploads/` 目录
2. **API 地址**: 默认 `http://localhost:3010/api`
3. **认证方式**: Bearer Token (admin/admin123)
4. **跨域支持**: 需确保后端 CORS 配置正确
