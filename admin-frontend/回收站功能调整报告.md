### 1️⃣ 导航栏调整
**移除**: 独立的"🗑️ 回收站"导航标签

**结果**: 导航栏从 8 个标签减少到 7 个
- 📊 数据统计

### 2️⃣ 商品管理页面新增按钮
在商品管理页面的操作按钮区域添加"🗑️ 回收站"按钮：

```html
<div style="margin-bottom: 16px; display: flex; gap: 10px; align-items: center;">
  <button class="btn btn-primary" onclick="showAddProduct()">+ 添加商品</button>
  <button class="btn btn-warning" onclick="showRecycleBin()">🗑️ 回收站</button>

```

### 3️⃣ 回收站弹窗
将回收站从独立页面改为**模态框弹窗**形式：

**弹窗结构**:
```
┌─────────────────────────────────────────────┐
│ 🗑️ 回收站                        [✕ 关闭]  │
├─────────────────────────────────────────────┤
│ [🔄 刷新] [✅ 批量恢复] [🗑️ 批量彻底删除]  │
├─────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────┐ │
│ │ 回收站商品列表（可滚动，最大高度 400px）│ │
│ └─────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

**特点**:
- ✅ 更宽的弹窗（900px 宽，95% 响应式）
- ✅ 列表区域可滚动（最大高度 400px）
- ✅ 顶部有关闭按钮
- ✅ 复用现有模态框样式

### 4️⃣ 新增函数

```javascript
// 显示回收站弹窗
function showRecycleBin() {
  document.getElementById('recycleBinModal').classList.add('active');
  loadRecycledProducts();
}

// 隐藏回收站弹窗
function hideRecycleBin() {
  document.getElementById('recycleBinModal').classList.remove('active');
}
```

### 5️⃣ 修改函数
- `switchTab()` - 移除 `recyclebin` 标签的处理逻辑

## 🚀 使用方式

1. **进入商品管理页面**
   - 点击顶部导航栏的"📦 商品管理"

2. **打开回收站**
   - 点击"🗑️ 回收站"按钮
   - 弹出回收站模态框

3. **管理已删除商品**
   - 查看已删除商品列表
   - 勾选商品进行批量操作
   - 点击"恢复"或"彻底删除"

4. **关闭回收站**
   - 点击右上角"✕ 关闭"按钮
   - 或点击弹窗外部区域

## ✨ 优化点

### 优点
- ✅ **更直观**: 回收站作为商品管理的一部分，逻辑更清晰
- ✅ **节省空间**: 减少导航标签数量，界面更简洁
- ✅ **操作便捷**: 在商品管理页面直接访问回收站，无需切换标签
- ✅ **上下文保持**: 管理商品和回收站之间切换更流畅

### 用户体验提升
- **场景 1**: 删除商品后，可以直接点击回收站按钮查看
- **场景 2**: 恢复商品后，关闭弹窗即可看到更新的商品列表
- **场景 3**: 减少导航栏拥挤，更适合小屏幕
