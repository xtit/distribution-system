## 📋 API 接口

### 认证接口
- `POST /api/auth/register` - 用户注册
- `POST /api/auth/login` - 用户登录
- `GET /api/auth/me` - 获取当前用户信息
- `GET /api/auth/referees` - 获取下级列表
- `GET /api/auth/commissions` - 获取佣金记录
- `GET /api/auth/referral-stats` - 获取分销统计

### 商品接口
- `GET /api/products` - 获取商品列表
- `GET /api/products/:id` - 获取商品详情
- `POST /api/products/orders` - 创建订单
- `GET /api/products/orders` - 获取订单列表
- `POST /api/products/orders/:id/payment` - 确认支付
- `POST /api/products/orders/:id/complete` - 完成订单

### 管理后台接口
- `GET /api/admin/dashboard` - 数据统计
- `GET /api/admin/users` - 用户列表
- `GET /api/admin/referral-tree` - 分销关系图
- `GET /api/admin/commissions` - 佣金列表
- `POST /api/admin/commissions/pay` - 发放佣金
- `GET /api/admin/orders` - 订单列表

## 🔧 技术栈

- **后端**: Node.js + Express + Sequelize + SQLite
- **前端**: Vue 3 + Vant UI + Pinia + Vue Router
- **认证**: JWT
- **数据库**: SQLite (可切换到 MySQL)
