## 📸 下一步：添加截图

### 方法 1: 快速截图（推荐）

1. **访问系统**:
   - 移动端：https://distribution-system-psi.vercel.app
   - 管理后台：https://distribution-system-admin.vercel.app

2. **使用浏览器截图**:
   - Chrome/Edge: `F12` → `Ctrl+Shift+P` → `Capture full size screenshot`
   - Safari: `Cmd+Shift+4` → 选择区域
   - Firefox: `F12` → 截图按钮

3. **保存截图**:
   ```bash
   # 保存文件到 docs/screenshots/ 目录
   # 按照命名规范命名文件
   ```

4. **提交到 Git**:
   ```bash
   git add docs/screenshots/*.png
   git commit -m "添加系统截图"
   git push
   ```

### 方法 2: 使用手机截图

1. 在手机上访问移动端
2. 访问各个页面并截图
3. 传输到电脑
4. 重命名并放到 `docs/screenshots/` 目录
5. 提交到 Git
