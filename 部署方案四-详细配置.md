## 🔧 第二步：配置阿里云服务器

### 1. 连接服务器

**方式一：阿里云控制台**
1. 登录阿里云控制台
2. 进入"轻量应用服务器"
3. 点击"远程连接"
4. 使用 VNC 或 Workbench 连接

**方式二：本地 SSH**
```bash
ssh root@你的服务器 IP
# 输入密码
```

### 2. 安装 Node.js 环境

```bash
# 更新系统
apt update && apt upgrade -y

# 安装 Node.js 20.x
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs

# 验证安装
node -v  # 应该显示 v20.x.x
npm -v   # 应该显示 10.x.x

# 安装 PM2（进程管理）
npm install -g pm2
```

### 3. 安装 Git

```bash
apt install -y git
git --version
```

### 4. 配置防火墙

```bash
# 开放必要端口
ufw allow 22    # SSH
ufw allow 3010  # 后端 API
ufw enable

# 查看状态
ufw status
```

### 5. 创建应用目录

```bash
# 创建应用目录
mkdir -p /var/www/distribution-system
cd /var/www/distribution-system

# 设置权限
chown -R $USER:$USER /var/www/distribution-system
```

## 🔒 第七步：配置 HTTPS（推荐）

### 使用 Nginx 反向代理

```bash
# 安装 Nginx
apt install -y nginx

# 创建 Nginx 配置
nano /etc/nginx/sites-available/distribution
```

**Nginx 配置**:
```nginx
server {
    listen 80;
    server_name 你的服务器 IP;

    location /api/ {
        proxy_pass http://localhost:3010/;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

**启用配置**:
```bash
# 创建软链接
ln -s /etc/nginx/sites-available/distribution /etc/nginx/sites-enabled/

# 测试配置
nginx -t

# 重启 Nginx
systemctl restart nginx
```

### 使用 Certbot 获取免费 HTTPS 证书

```bash
# 安装 Certbot
apt install -y certbot python3-certbot-nginx

# 获取证书（需要域名）
certbot --nginx -d your-domain.com

# 自动续期
certbot renew --dry-run
```

## 🔧 自动化部署脚本

### 本地部署脚本

```bash
#!/bin/bash
# deploy.sh

SERVER_IP="你的服务器 IP"
SERVER_USER="root"

echo "🚀 开始部署..."

# 1. 构建前端
echo "📦 构建前端..."
cd mobile-frontend
npm install
npm run build
cd ..

# 2. 上传后端
echo "📤 上传后端..."
tar -czf backend.tar.gz backend/
scp backend.tar.gz ${SERVER_USER}@${SERVER_IP}:/var/www/distribution-system/
rm backend.tar.gz

# 3. SSH 执行部署
echo "🔧 远程部署..."
ssh ${SERVER_USER}@${SERVER_IP} << 'ENDSSH'
cd /var/www/distribution-system/backend
tar -xzf ../backend.tar.gz
npm install
pm2 restart distribution-backend
ENDSSH

# 4. 部署前端到 Cloudflare
echo "☁️ 部署到 Cloudflare..."
cd mobile-frontend/dist
# 手动上传到 Cloudflare Pages 或使用 wrangler CLI

echo "✅ 部署完成！"
echo "前端：https://distribution-frontend.pages.dev"
echo "后端：http://${SERVER_IP}:3010/api"
```
