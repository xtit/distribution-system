## 🎨 美化内容

### 1️⃣ 商品列表页 (Home.vue)

#### 整体风格
- ✅ 渐变背景：`linear-gradient(180deg, #f7f8fa 0%, #fff 100%)`
- ✅ 最小高度：`min-height: 100vh`
- ✅ 卡片式设计，圆角 12px

#### 商品卡片
**美化前**:
```css
.product-item {
  background: #fff;
  border-radius: 8px;
  padding: 10px;
}
```

**美化后**:
```css
.product-item {
  background: #fff;
  border-radius: 12px;
  padding: 12px;
  margin-bottom: 12px;
  box-shadow: 0 2px 12px rgba(0, 0, 0, 0.04);
  transition: all 0.3s ease;
}

.product-item:active {
  transform: scale(0.98);
  box-shadow: 0 1px 6px rgba(0, 0, 0, 0.08);
}
```

**效果**:
- ✅ 圆角更大（8px → 12px）
- ✅ 添加阴影效果
- ✅ 点击动画（缩放 0.98）
- ✅ 过渡动画（0.3s）

#### 商品图片
**美化前**:
```css
width="100" height="100" round
```

**美化后**:
```css
:deep(.van-image) {
  border-radius: 12px;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.06);
}
```

**效果**:
- ✅ 圆角与卡片一致
- ✅ 添加阴影
- ✅ 防止变形

#### 价格显示
**美化前**:
```css
.current-price {
  font-size: 18px;
  color: #f44;
  font-weight: bold;
}
```

**美化后**:
```css
.current-price {
  font-size: 20px;
  color: #ff4757;
  font-weight: 700;
  font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;

```

**效果**:
- ✅ 字体更大（18px → 20px）
- ✅ 颜色更鲜艳（#f44 → #ff4757）
- ✅ 使用系统字体
- ✅ 添加价格基线对齐
