## 🐛 问题描述

用户反馈：移动端商品列表页和详情页都没有显示商品图片。

## 🔍 问题分析

### 1️⃣ 端口错误
我之前误以为移动端是 3011 端口（实际正确），但测试时用了错误的端口。

**移动端正确端口**: **3011**

### 2️⃣ Vite 代理配置缺失

**问题**: `vite.config.js` 中缺少 `/uploads` 路径的代理配置

**修复前**:
```javascript
proxy: {
  '/api': {
    target: 'http://localhost:3010',
    changeOrigin: true
  },
  '/qrcodes': {
    target: 'http://localhost:3010',
    changeOrigin: true
  }
}
```

**问题**: 图片 URL 是 `/uploads/image-xxx.png`，没有代理配置，导致 404

### 3️⃣ 图片 URL 处理

后端返回的图片 URL 是相对路径：
```json
{
  "mainImage": "/uploads/image-1774060611340-212405546.png"
}
```

移动端需要正确处理这个 URL，通过 Vite 代理访问后端静态文件。

## ✅ 修复方案

### 1️⃣ 添加 /uploads 代理

**文件**: `mobile-frontend/vite.config.js`

```javascript
server: {
  port: 3011,
  proxy: {
    '/api': {
      target: 'http://localhost:3010',
      changeOrigin: true
    },
    '/uploads': {  // ✅ 新增
      target: 'http://localhost:3010',
      changeOrigin: true
    },
    '/qrcodes': {
      target: 'http://localhost:3010',
      changeOrigin: true
    }
  }
}
```

**原理**: 
- 前端请求 `/uploads/image-xxx.png`
- Vite 代理转发到 `http://localhost:3010/uploads/image-xxx.png`
- 后端返回静态文件

### 2️⃣ 添加图片 URL 处理函数

**文件**: `mobile-frontend/src/views/Home.vue` 和 `ProductDetail.vue`

```javascript
// 获取图片完整 URL
const getImageUrl = (url) => {
  if (!url) return ''
  // 如果已经是完整 URL，直接返回
  if (url.startsWith('http://') || url.startsWith('https://') || url.startsWith('data:')) {
    return url
  }
  // 相对路径，通过 Vite 代理访问
  return url
}
```

**使用**:
```vue
<!-- 修复前 -->
<van-image :src="product.mainImage || 'https://via.placeholder.com/200'" />

<!-- 修复后 -->
<van-image :src="getImageUrl(product.mainImage) || 'https://via.placeholder.com/200'" />
```

## 🔧 故障排查

### 图片仍然不显示？

**检查 1**: 后端服务是否运行
```bash
curl http://localhost:3010/uploads/image-xxx.png
# 应该返回图片内容
```

**检查 2**: Vite 代理是否生效
```bash
curl http://localhost:3011/uploads/image-xxx.png
# 应该返回图片内容
```

**检查 3**: 数据库中的图片路径
```bash
curl -H "Authorization: Bearer TOKEN" \
  http://localhost:3010/api/products/1
# 检查 mainImage 字段是否有值
```

**检查 4**: 浏览器控制台错误
- Network 标签查看图片请求状态
- Console 标签查看错误信息

### 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| 404 Not Found | 代理配置缺失 | 添加 `/uploads` 代理 |
| 403 Forbidden | CORS 问题 | 检查后端 CORS 配置 |
| 图片空白 | mainImage 为 null | 运行更新脚本 |
| 端口冲突 | 3011 被占用 | `lsof -i :3011` 检查 |
