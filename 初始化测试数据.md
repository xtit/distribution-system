# 📦 测试数据初始化脚本

这个脚本会在 Railway 上执行，添加测试商品数据到数据库。

## 执行步骤

1. 登录 Railway Dashboard
2. 进入项目：positive-liberation
3. 点击服务：distribution-system
4. 点击 "Deployments" 标签
5. 找到最新的部署
6. 点击 "View logs"
7. 在 Console 中执行：`node src/scripts/init-db.js`

或者直接让代码在启动时自动执行。
