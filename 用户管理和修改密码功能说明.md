## 📱 使用流程

### 修改密码（管理后台）

1. **登录管理后台**: https://distribution-system-admin.vercel.app
2. **进入系统配置**: 点击"系统配置"标签
3. **填写密码信息**:
   - 原密码：当前使用的密码
   - 新密码：至少 6 位
   - 确认新密码：与新密码一致
4. **点击确认修改**: 验证通过后提交
5. **修改成功**: 自动退出登录，需要重新登录

### 禁用/启用用户

1. **进入用户管理**: 点击"用户管理"标签
2. **查看用户列表**: 所有用户信息
3. **点击状态按钮**:
   - 正常用户显示"禁用"按钮
   - 禁用用户显示"启用"按钮
4. **确认操作**: 弹窗确认
5. **立即生效**: 用户状态更新

### 重置用户密码

1. **进入用户管理**
2. **找到目标用户**
3. **点击"重置密码"**
4. **输入新密码**: 默认 123456
5. **确认重置**: 立即生效

### 删除用户

1. **进入用户管理**
2. **找到目标用户**（非管理员）
3. **点击"删除"**
4. **二次确认**: 警告不可恢复
5. **确认删除**: 永久删除

## 🔧 技术实现

### 后端 API

#### 修改密码
```javascript
POST /api/auth/change-password
Headers: Authorization: Bearer {token}
Body: {
  "oldPassword": "原密码",
  "newPassword": "新密码"
}

Response: {
  "code": 200,
  "message": "密码修改成功，请重新登录"
}
```

#### 获取用户列表
```javascript
GET /api/admin/users
Headers: Authorization: Bearer {token}

Response: {
  "code": 200,
  "data": {
    "list": [
      {
        "id": 1,
        "username": "admin",
        "nickname": "系统管理员",
        "phone": "13800000000",
        "referralCode": "ADMIN001",
        "referrer": null,
        "balance": "0.00",
        "status": 1
      }
    ]
  }
}
```

#### 更新用户状态
```javascript
PUT /api/admin/users/{id}/status
Headers: Authorization: Bearer {token}
Body: {
  "status": 0  // 0-禁用，1-正常
}
```

#### 重置用户密码
```javascript
PUT /api/admin/users/{id}/reset-password
Headers: Authorization: Bearer {token}
Body: {
  "password": "新密码"
}
```

#### 删除用户
```javascript
DELETE /api/admin/users/{id}
Headers: Authorization: Bearer {token}
```

### 前端实现

#### 修改密码
```javascript
async function changePassword() {
  const oldPassword = document.getElementById('oldPassword').value;
  const newPassword = document.getElementById('newPassword').value;
  const confirmPassword = document.getElementById('confirmPassword').value;
  
  // 验证
  if (!oldPassword || !newPassword || newPassword.length < 6) {
    showToast('密码格式错误', 'error');
    return;
  }
  
  if (newPassword !== confirmPassword) {
    showToast('两次密码不一致', 'error');
    return;
  }
  
  // API 调用
  const res = await fetchWithAuth(`${API_BASE}/auth/change-password`, {
    method: 'POST',
    body: JSON.stringify({ oldPassword, newPassword })
  });
  
  if (res.code === 200) {
    // 清除登录信息并跳转
    localStorage.removeItem('admin_token');
    localStorage.removeItem('admin_user');
    window.location.href = 'login.html';
  }
}
```

#### 用户状态切换
```javascript
async function toggleUserStatus(userId, currentStatus) {
  const newStatus = currentStatus === 1 ? 0 : 1;
  const actionText = newStatus === 1 ? '启用' : '禁用';
  
  showModal(
    `确认${actionText}用户`,
    `确定要${actionText}该用户吗？`,
    async () => {
      const res = await fetchWithAuth(`${API_BASE}/admin/users/${userId}/status`, {
        method: 'PUT',
        body: JSON.stringify({ status: newStatus })
      });
      
      if (res.code === 200) {
        showToast(`${actionText}成功`, 'success');
        await loadUsers();
      }
    }
  );
}
```

## 📊 数据库字段

### users 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |

| totalCommission | DECIMAL(10,2) | 总佣金 |
